These files are for use with packetView and bitView. While packetView can open and display .bin files (using the raw input option) they are best viewed in bitView. The same goes for pcap files, they can be viewied in bitView, however bitView will just display the file as a raw binary stream, it will not make use of the pcap header to break up the file into packets

Note that these files were created in a simulated environment. They were artificially crafted for the purposes of development and training. The Author gives consent for these files to be used/viewed/modified/analysed in any way, by anyone, for any purpose.
